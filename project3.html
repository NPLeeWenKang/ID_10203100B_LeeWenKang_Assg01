ColorPicker Game
Website / Github
The first ever website that I created was a portfolio website. I would follow a tutorial word by word. However after creating my first website, I wanted to build a website completely from scratch. So I decieded to create a game on a website. As this was the first website I created completely by my own (not following tutorial), I encountered many problems. However after diligently looking for answers online and testing the solutions, I would always figure out the solution by trial and error.
As I encountered many errors and was unsure about alot of things, I had to google alot. As a result, this project has developed my problem solving skills. This means I developed myself greatly as a programmer as I learnt how to adapt online solutions to cater to my needs and changed my outlook on programming from "just copying code even if I don't understand" to "understand the code, then change it to make it cater to your needs". I also learnt how to properly incorporate and understand other people's code.
